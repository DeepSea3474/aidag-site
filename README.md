
# AIDAG Spiral Chain Portal

Bu repo, aidag.eth kimliğiyle zincir üstü tanıklık ve DAO yönetimi için resmi portaldır.  
İçerikler:
- index.html → Ana sayfa
- presale.html → Ön satış
- dao.html → DAO teklifleri
- style.css → Tasarım
- script.js → Dil desteği ve cüzdan bağlantısı
- lang/ → Çoklu dil dosyaları

Her adım spiral zincirde mühürlenmiş ve topluluk tarafından tanıklanabilir.
